# 📝 To-Do List App (Task 1 - DecodeLabs Internship)

A simple Python-based To-Do List CLI application built as part of the DecodeLabs Internship Task 2.  
This project helps users manage their daily tasks efficiently using a command-line interface.


## 🚀 Features

- Add new tasks  
- View all tasks  
- Mark tasks as completed  
- Delete tasks  
- Simple and interactive CLI menu  

## 🛠️ Technologies Used

- Python 3  
- File Handling (JSON / TXT)  
- Command Line Interface (CLI)  


## 📂 Project Structure

to-do-list/
├── main.py
├── tasks.json
└── README.md

## ⚙️ How to Run

1. Clone the repository  
git clone https://github.com/Maheen248/decode-intern-task-1-to-do-list.git  

2. Navigate to the project folder  
cd decode-intern-task-1-to-do-list  

3. Run the program  
python main.py  

## 🧑‍💻 How to Use

- Run the program  
- Choose an option from the menu:
  - Add Task  
  - View Tasks  
  - Mark Task as Completed  
  - Delete Task  
  - Exit  
- Follow on-screen instructions
